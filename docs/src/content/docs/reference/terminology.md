---
title: Terminology
description: Glossary of terms used in Talkdraft.
---

| Term | Meaning |
|------|---------|
| **Capture** | A raw brain dump â€” either voice or text. The act of creating a note. |
| **Note** | A single item in the app. Has content (text), an optional title, optional audio, and optional category assignment. |
| **Category** | A user-defined organizational bucket with a name and color (e.g., Ideas, Tasks, Journal). |
| **Transcript** | The text output from Groq Whisper after processing a voice recording. |
| **Rewrite** | AI-powered content transformation using Gemini Flash. Applies a tone or custom instructions to restyle the note text. |
| **Append** | Adding a new voice recording to an existing note. The transcript is inserted at the cursor position. Audio is not saved. |
| **Soft Delete** | Moving a note to Recently Deleted rather than permanently removing it. Notes are auto-purged after 30 days. |
| **Pro** | The paid subscription tier. Unlocks 15-minute recordings, unlimited notes, and unlimited categories. |
| **Tone** | A rewrite style preset (e.g., "Clean up", "Flirty", "Formalize"). 17 built-in tones plus custom instructions. |
| **FFT Visualization** | Real-time frequency analysis displayed as animated bars during voice recording. |
| **Edge Function** | A serverless function running on Supabase that proxies requests to external AI APIs (Groq, Gemini). |
| **Liquid Glass** | Apple's iOS 26 design language featuring translucent, glass-like UI elements. |
