## Purpose

The settings screen provides access to preferences, utility tools, and account management.

## Sections

### General

- **Language** — sets the app's display language
- **Theme** — light, dark, or system (follows device setting)

### Tools

- **Import Audio** — opens the system file picker to import an audio file (`.m4a`, `.mp3`, `.wav`, `.aac`, `.caf`). The imported file goes through the same transcription pipeline as a voice recording, creating a new note with the transcript.

### Recently Deleted

- Shows a count of soft-deleted notes
- Tapping opens the [Recently Deleted](/views/recently-deleted/) screen
- Notes remain in the trash for 30 days before automatic permanent deletion

### Legal

- **Privacy Policy** — opens in Safari
- **Terms of Use** — opens in Safari

### Account

- **Sign out** — signs the user out and returns to the login screen
- **Delete account** — schedules account deletion with a 30-day grace period. During the grace period, a red warning banner appears in Settings with an option to cancel deletion.
